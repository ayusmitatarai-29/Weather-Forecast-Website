# Weather-Forecast-Website
A responsive Weather Forecast Website built using HTML, CSS, and JavaScript that provides real-time weather updates and a 5-day forecast for any city in the world.

# Features 
1. Auto- Detect user live location 

2. Search any city worldwide 

3. Real time temprature and wind Speed

4. 5 Day weather forecast with daily cards 

5. Automatic Day/Night Detection with local time

6. Live interactive Map

7. Dark mode/ Light mode

8. Fully Responsive Design 

# Technologies Used

1. HTML5

2. CSS3

3. JavaScript 

4. OpenWeather API

5. OpenStreetMap
